Anime and Games

This API uses a list of Anime and games as its resource. The routes are as follows: tr class="border-2">
Route Method Description Example 
/anime
 GET Gets a list of anime and ids List anime 
/anime/:id
 GET Returns the detail about the anime matching the id in the parameter Get anime 1 
/game
 GET Gets a list of games and ids List game 
/game/:id
 GET Returns the detail about the game matching the id in the parameter Get game 1
